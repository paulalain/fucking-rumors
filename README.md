
Fucking Rumors
========

Prerequisites:

  * Running MongoDB instance: `mongod --dbpath /path/to/db`
  * Node.js >= 0.12
  * Ruby
  * Sass (install 'gem install sass')
  * Grunt (install: `npm install -g grunt-cli`)

Install: `npm install`

Start:

  * Server: `npm start`
  * Client resources: `grunt ; grunt watch`
